# WhatsApp Style Web Chat App

A beautiful WhatsApp-style web chat app with:

- Two-person chat (Person A = Me, Person B = AI Bot)
- Editable messages
- Timestamps
- Local storage for messages
- Export chat to PDF with chat bubbles
- Fully responsive (tablet/laptop-friendly)
- Deployable on GitHub Pages

## How to Use

1. Select sender (Person A or Person B)
2. Type a message and click send
3. Click a message to edit
4. Export PDF using the "Export PDF" button

## Deployment on GitHub Pages

1. Create a GitHub repository and add all files (`index.html`, etc.)
2. Go to **Settings → Pages**
3. Select **Branch: main** and folder: `/ (root)` → Save
4. Your app will be live at:  
   `https://<username>.github.io/<repo-name>/`
